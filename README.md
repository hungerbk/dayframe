# dayframe

하루 일정을 원형 타임테이블로 직관적으로 계획하고 시각화하는 웹 앱입니다.
사용자는 시간 블록을 추가하고 편집하며, 다양한 테마와 스케치 스타일을 적용해
나만의 하루 일정을 쉽게 만들 수 있습니다.

## 목표

이 프로젝트는 개인적으로 최근 생활 패턴이 흐트러져 이를 개선하려는 목적에서 시작했습니다. 원래는 슬랙 등 알림 연동을 고민했지만, 로그인·백엔드 기능이 필요하면 사용 진입 장벽이 커진다고 판단해 우선 프론트엔드 중심으로만 구현했습니다.

사용자가 별도 가입 없이도 빠르게 하루를 계획하고 꾸며 결과물을 저장할 수 있도록 하는 것이 핵심 목표입니다.

## 해결하려는 문제

- 일과 생활 패턴이 불규칙한 사용자가 시각적으로 쉽게 계획을 세우고 기억하게 함
- 불필요한 로그인이나 복잡한 설정 없이 바로 사용할 수 있는 간단한 사용성
- 시각적으로 꾸밀 수 있는 결과물을 만들어 폰 배경이나 다이어리 이미지로 활용할 수 있게 함

## 구현 방향

- 프론트엔드 단독 구현: 빠른 사용성과 낮은 진입 장벽을 위해 백엔드·로그인 미탑재
- 불필요한 라이브러리 최소화: 상태관리 라이브러리나 무거운 의존성 없이 간단한 상태와 로컬 저장으로 동작
- 이미지 내보내기(PNG)와 테마/스케치 스타일 제공: 사용자가 결과물을 다양하게 활용하도록 설계

## 업데이트 내역

- 현재(v1.0.2): 모바일 UI/UX 개선을 중점적으로 진행해 휴대폰에서의 사용성을 높였습니다.
- 향후 계획: 타임트래커 기능(활동 기록, 통계 등)을 추가할 예정입니다. 필요에 따라 소셜 공유도 추가할 예정입니다.

## 실행 방법

Node.js 18 이상이 필요합니다.

```bash
npm install
npm run dev
```

## 배포

- 사이트: https://dayframe.pages.dev/

## 주요 기능

- 하루 24시간을 원형 타임테이블로 시각화
- 시작 시간과 종료 시간을 입력해 일정 블록 생성
- 생성된 블록 편집 및 삭제 기능
- 테마 선택과 스케치 스타일 전환으로 분위기 변경
- 시간 눈금 표시 옵션으로 원하는 정보만 보이기
- PNG 이미지로 다운로드하고 배경 투명화 옵션 사용 가능
- 작업 상태를 자동 저장하여 새로고침 후에도 일정 유지
